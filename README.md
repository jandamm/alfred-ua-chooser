# Alfred Universal Action Chooser
Alfred universal action to select items

This is a small workflow which works for multiple items and allows you to select any item to further process with universal actions.

## Actions:
- ↩ Choose current file
- ⌘↩ Remove current item (choose everything else)
- ⌥↩ Choose as url with https (only available if url is detected)
- ⇧⌥↩ Choose as url with http (only available if url is detected)

It works with mixed items as well as uniform items.
